# Blog Web Application

## Overview

A simple, modern blog web application that allows users to create, view, edit, and delete posts. Posts do not persist between sessions.

## Features

- **Post Creation**: Users can create posts with a title, content, and an image URL.
- **Post Viewing**: Posts are displayed in a gallery view, with the newest posts appearing first.
- **Post Update/Delete**: Users can edit or delete posts directly from the home page.
- **Styling**: The app is well-styled, responsive, and includes a dark mode.
